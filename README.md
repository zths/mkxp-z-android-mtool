# 用于 Android 的 mkxp-z + mtool

此项目是 [mkxp-z-android](https://github.com/thehatkid/mkxp-z-android) 的一个分支版本，用于 MTool 中。

[mkxp-z](https://github.com/mkxp-z/mkxp-z) 的 Android 移植版本。

大量使用了 AI 代码, 质量可叹.
